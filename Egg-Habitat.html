waitTilNextFilled :: forall habs. KnownNat habs => HabData habs -> Bonuses -> IsCalm -> HabStatus habs -> Either WaitError ((Fin N4, Double), (HabStatus habs, Double)) Source #

Calculate the time until the next hab is full, and return the updated habs after that time. Returns Nothing if all habs are full.

Also returns the current total fill rate per hatchery, taking into account all bonuses.

Conclusions:

  1. No internal hatcheries
  2. All habs are already full.
  3. Hab n filled after a given amount of time.
